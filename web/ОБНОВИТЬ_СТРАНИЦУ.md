# Как увидеть последние изменения

1. **Закрой все вкладки** с приложением (Kids Motivation).

2. **Останови старые dev-серверы:** в терминале Cursor закрой вкладки, где запущен `npm run dev`, или выполни:
   ```bash
   lsof -ti:5173,5174,5175,5176,5177 | xargs kill -9 2>/dev/null; echo "Готово"
   ```

3. **Запусти один раз фронтенд:**
   ```bash
   cd "/Users/mishashubin/Desktop/Work /Children/web"
   npm run dev
   ```

4. **Открой в браузере только один адрес** — тот, что покажет Vite (например `http://127.0.0.1:5173/`).

5. **Жёсткое обновление:** `Cmd + Shift + R` (Mac) или `Ctrl + Shift + R` (Windows).

6. **Проверь:**
   - Войди как **ребёнок** → раздел **Задания**. Справа от заголовка «Задания на сегодня» должна быть синяя кнопка **«Обновить задания»**.
   - Блок **«Моя цель»** / **«Избранное желание»** — белая карточка, фото слева, цифры справа.

Если кнопки «Обновить задания» нет — в DevTools (F12) → вкладка **Network** → включи **Disable cache** и снова сделай `Cmd + Shift + R`.
