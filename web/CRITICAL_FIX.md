## 🔥 ПОЛНАЯ ОЧИСТКА КЕША (ОБЯЗАТЕЛЬНО!):

### Способ 1: Жесткая перезагрузка
- **Mac**: `Cmd + Shift + R` (или `Cmd + Option + R`)
- **Windows/Linux**: `Ctrl + Shift + R` (или `Ctrl + F5`)

### Способ 2: Через DevTools (РЕКОМЕНДУЕТСЯ)
1. Откройте DevTools: **F12** или **Cmd+Option+I** (Mac) / **Ctrl+Shift+I** (Windows)
2. Перейдите на вкладку **Network**
3. **ВКЛЮЧИТЕ** опцию **"Disable cache"** (чекбокс вверху)
4. **ОСТАВЬТЕ DevTools открытым**
5. Обновите страницу: **F5** или **Cmd+R** / **Ctrl+R**

### Способ 3: Полная очистка кеша браузера

#### Chrome/Edge:
1. Нажмите `Cmd+Shift+Delete` (Mac) или `Ctrl+Shift+Delete` (Windows)
2. Выберите "Кэшированные изображения и файлы"
3. Выберите "За все время"
4. Нажмите "Очистить данные"
5. **Закройте все вкладки браузера**
6. **Перезапустите браузер**

#### Firefox:
1. Нажмите `Cmd+Shift+Delete` (Mac) или `Ctrl+Shift+Delete` (Windows)
2. Выберите "Кеш"
3. Выберите "Все"
4. Нажмите "Очистить сейчас"
5. **Закройте все вкладки браузера**
6. **Перезапустите браузер**

### Способ 4: Режим инкогнито (для проверки)
1. Откройте новое окно в режиме инкогнито:
   - **Mac**: `Cmd+Shift+N`
   - **Windows**: `Ctrl+Shift+N`
2. Откройте приложение
3. Если в инкогнито все работает - проблема точно в кеше

## ✅ Проверка после очистки:

1. Откройте DevTools (F12) → **Console**
2. **НЕ должно быть** ошибки "Rendered more hooks than during the previous render"
3. Если ошибка осталась:
   - Откройте DevTools → **Sources**
   - Найдите `ChildSwitcher.tsx`
   - Проверьте, что в файле **НЕТ** хуков (`useState`, `useEffect`, `useMemo`, `useCallback`)
   - Проверьте, что есть строка `// ВЕРСИЯ: 2.1.0`

## 🐛 Если проблема все еще есть:

1. **Проверьте версию файла в браузере:**
   - DevTools → Sources → `ChildSwitcher.tsx`
   - Должна быть строка: `// ВЕРСИЯ: 2.1.0`

2. **Проверьте консоль на другие ошибки:**
   - Может быть проблема в другом компоненте

3. **Попробуйте другой браузер:**
   - Если в другом браузере работает - проблема в кеше первого

4. **Полная переустановка зависимостей:**
   ```bash
   cd web
   rm -rf node_modules package-lock.json
   npm install
   npm run dev
   ```
